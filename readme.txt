垂直分库
把不同的表按照逻辑不同的模块，分到不同的数据库中

垂直分表
把一个表的不同列划分到不同的表中

把数据库创建相同数据的数据库，表也相同结构。但是，例如对于 student 表，
id % 2 == 0 就去 DB1_Table; id % 2 == 1 就去 DB2_table


垂直是改变结构的
水平是改变数量的

PS1: 在设计的时候就要考虑到水平或者垂直拆分
PS2: 首先考虑缓存处理，读写分离，使用索引，然后再考虑垂直分表或者水平分表
